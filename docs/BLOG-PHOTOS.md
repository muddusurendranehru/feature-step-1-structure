# Blog photos — HOMA Health Blog

Add featured images for each article by placing image files in **`public/blog/`** with these exact filenames. If a file is missing, the blog card will show the placeholder automatically.

## Required filenames (save as JPG or PNG)

| Filename | Article |
|----------|---------|
| `acv.jpg` | Apple Cider Vinegar for Metabolism & Blood Sugar Control |
| `honey-vs-sugar.jpg` | Honey vs Sugar: Which is Better for Diabetes & Metabolism? |
| `almonds-pcos.jpg` | Why Almonds Are a Superfood for Insulin Resistance & PCOS |
| `cinnamon.jpg` | Cinnamon for Glucose Control — How Much & Which Type? |
| `intermittent-fasting.jpg` | Intermittent Fasting for Metabolism Boost — Safe or Risky? |
| `dr-muddu-mvp-blog.jpg` | Dr. Muddu MVP Blog (external blog card) |

## Suggested images (from your assets)

You can use your existing graphics and save them with the names above:

- **almonds-pcos.jpg** — Use the “almonds diabetes: Evidence-Based Guide” infographic (almonds + diabetes content).
- **intermittent-fasting.jpg** — Use “Prevent Type 2 Diabetes” or “How to Control Diabetes” (metabolism/diabetes visuals).
- **dr-muddu-mvp-blog.jpg** — Use the Dr. Muddu MVP Blog / HOMA Diagnostics promo with Dr. Surendra Nehru, or the “How to Control Diabetes” graphic with Dr. M. Surendra Nehru.
- **acv.jpg** — Use “Healthy Lifestyle Program” (nutrition/wellness) or any ACV/food image.
- **honey-vs-sugar.jpg** — Any nutrition/sugar/diabetes image.
- **cinnamon.jpg** — Any spice/nutrition or glucose-control image.

## Steps

1. Resize/crop images to roughly **600×400** or similar aspect ratio for consistent cards.
2. Save (or copy) each file into `public/blog/` with the exact filename from the table.
3. Refresh `/blog` — cards will show your photos; missing files will still show the “Blog image” placeholder.

No code changes are needed when you add or replace images.
